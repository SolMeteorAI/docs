**‚Üí Pool structure (DLMM or DAMM)**

Each pool is ranked using MeteorScore, Fabriq's proprietary activity-based scoring model.

MeteorScore emphasizes real usage over static yield, helping highlight pools with meaningful participation rather than temporary spikes.

## Security Badges

Trending Pools displays security badges directly in the table to surface important token characteristics at a glance.

Common badges include:

**‚Üí Freezable Token** ‚Äî the token contract can freeze transfers  
**‚Üí Mintable Token** ‚Äî the token supply can be expanded  
**‚Üí Jupiter Verified** ‚Äî the token has been verified by Jupiter Exchange

Badges are intended as signals, not guarantees, and should be considered alongside other market indicators.
