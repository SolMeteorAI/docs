title: Filters & Metrics

## Search

Pools can be searched using:

- Pool name (e.g. SOL-USDC)
- Token symbol
- Token address

Search works alongside all filters.

## Pool Structure

**LP Type** — filter by pool structure

## Token Controls

- **Include Tokens** — require specific token symbols or addresses
- **Exclude Tokens** — remove pools containing selected tokens

## Activity & Scoring Filters

- Fabriq Score (Min / Max)
- Trend
- Token Age (minutes)
- Pool Age (minutes)

## Liquidity, Valuation & Participation

- Pool Liquidity (Min / Max)
- Token Total Liquidity (Min / Max)
- FDV (Min / Max)
- Holders (Min / Max)

## Fee Filters

- LP Fees 5m (Min / Max)
- LP Fees 1h (Min / Max)
- LP Fees 6h (Min / Max)
- LP Fees 24h (Min / Max)

## APR Filters

- APR 5m (Min / Max)
- APR 1h (Min / Max)
- APR 6h (Min / Max)
- APR 24h (Min / Max)

## Volume Filters

- Volume 5m (Min / Max)
- Volume 1h (Min / Max)
- Volume 6h (Min / Max)
- Volume 24h (Min / Max)

## Transaction Filters

- Transactions 5m (Min / Max)
- Transactions 1h (Min / Max)
- Transactions 6h (Min / Max)
- Transactions 24h (Min / Max)

## Rate-Based Metrics

- Average swaps per minute (Min / Max)
- Average volume per minute (Min / Max)
- Average fees per minute (Min / Max)

## Column Controls

Users can add or remove columns for:

- Fabriq Score
- Trend
- Age
- FDV
- Liquidity
- Holders
- LP Fees (5m, 1h, 6h, 24h)
- Price Change (5m, 1h, 6h, 24h)
- Volume (5m, 1h, 6h, 24h)
- Transactions (5m, 1h, 6h, 24h)
- APR (5m, 1h, 6h, 24h)
- Makers (5m, 1h, 6h, 24h)
- Average swaps, volume, and fees per minute

Layouts can be reset to default at any time.
